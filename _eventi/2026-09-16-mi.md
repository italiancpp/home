### La sessione

**Functional Programming in C++: from a Scala perspective**

C++ is presented as a multi-paradigm language — but how does it actually fare when you put it to the functional programming test? In this talk we explore the core ideas of functional programming and map them onto C++, using Scala as a touchstone: a language with genuine, first-class FP support. We'll see where C++ holds its own, where it falls short, and — most importantly — which FP principles you can apply in your C++ codebase starting Monday morning.

### Il relatore

**Massimiiliano Pagani** 

Max started coding in the 80s, when computers were laughably slow, never had enough RAM, and used tape recorders for storage. Since then he has never stopped writing bugs and other features. After years in the video game industry he ended up in firmware and embedded software — which is how C++ became a daily companion — and somewhere along the way got genuinely fascinated by functional programming. When something strikes him as interesting and he has enough spare time, he writes about it at maxpagani.org.
